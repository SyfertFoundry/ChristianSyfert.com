## About

I build AI systems that do real work: fine-tuned models that classify documents with 93% accuracy, agent loops that hold natural conversations, and pipelines that turn unstructured data into structured decisions.

Most recently, I was the primary engineer on an ML-powered document intelligence system built for a Fortune 500 enterprise client. I designed and delivered the full stack: a fine-tuned DeBERTa v3 model with multi-task learning and BIO tagging with CRF layers, a three-service API architecture, Azure Document Intelligence integration for OCR, a relational database layer, and the web frontend. I owned 90% of that project as a solo IC.

I'm also the founder of WhisperForm, a conversational form platform that replaces static lead capture with natural, AI-driven dialogue. The architecture is an agentic loop with strict-mode tool calling, JSON schema enforcement, and real-time UI rendering, built on TypeScript, Node.js, and the OpenAI API. I evaluated multiple LLM providers (GPT-5 Mini, Gemini, Granite) on schema reliability, latency, and constrained decoding before choosing the production stack. Currently in validation.

Before software, I was a registered counsellor in South Africa, where I founded a tele-therapy platform. That background shapes how I build: I think about how people actually interact with systems, not just how systems process data. It's why WhisperForm exists.

I develop entirely through the NVDA screen reader. Every architecture decision, every model evaluation, every deployment, built without sight. Accessibility isn't a compliance checkbox in my work. It's the foundation I build from.

I'm focused on agentic AI, applied ML, and conversational interfaces.
